== README

Toney is an app to help you track how much money you spent and on what exactly you spent.

* Ruby version - 2.1.2, Rails version - 4.1.4

* Before starting the app:
  
  <code>
  bin/bundle install
  </code>
  
  
  <code>
  rake db:migrate
  </code>
  
* run the test suite:

  <code>
  bin/rake
  </code>
